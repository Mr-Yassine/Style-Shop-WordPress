== Description ==

RaraTheme Companion is truly a must have companion for your website. It provides 14 useful and frequently used widgets that you can add to the sidebars to create an engaging website.

== Widgets offered ==

1. Client Logo
2. Call to Action
3. Faqs
4. Featured Page
5. Featured Post
6. Icon Text
7. Image
8. Popular Post
9. Recent Post
10. Social Links
11. Stat Counter
12. Team Member
13. Testimonial
14. Twitter Feeds

The plugin works best with themes developed by Rara Theme.

The plugin also provides simple yet clean code which allows developers to overwrite the plugin properties. The plugin offers a multiple numbers of filters and action hooks which for maximum flexibility to the developers.

== Installation ==

= Uploading in WordPress Dashboard =

1. Navigate to the 'Plugins> Add New' in the WordPress dashboard
2. Click on 'Upload' button.
3. Select raratheme-companion.zip from your computer
4. Click 'Install Now'
5. Activate the plugin from the Plugin dashboard

= Using FTP =

1. Download raratheme-companion.zip
2. Extract the raratheme-companion directory to your computer
3. Upload the raratheme-companion directory to the /wp-content/plugins/ directory
4. Activate the plugin in the Plugin dashboard
